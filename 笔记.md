git 分布式版本控制系统
从个人开发到团队协作

### 三大区域
    工作区、暂存区、本地库、远程库
    工作区 git add =>  暂存区
    暂存区 git commit =>  本地库

#### 工作区
    写代码，还未保存
#### 暂存区
    代码已经临时保存下来，但还没有提交上传到库中
#### 本地库
    代码最终保存的地方，作为历史版本（不能删除！！！）
#### 远程库
    代码托管中心，如github，gitee等，局域网：gitlab

#### 常用命令
    git config--global user.name 用户名          设置用户签名
    git config--global user.email 邮箱           设置用户签名
    git init                                    初始化本地库        （生成.git文件夹）
    git status                                  查看本地库状态
    git add 文件名                              添加至暂存区
    git commit -m '日志文件' 文件名             提交到本地库
    git reflog                                  查看历史记录（本地库）
    git reset --hard 版本号                           版本穿梭

    git rm --cached 文件名                      从暂存区中删除文件 
    git log                                     查看详细的历史记录
